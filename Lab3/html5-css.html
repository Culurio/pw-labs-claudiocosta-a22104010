HTML5 e CSS

HTML 5

HTML é a linguagem de marcação padrão para páginas da Web.

header ✔ indica cabeçalho
article ❌ indica porção do documento que pode ser vista de forma independente
nav ✔ indica menu de navegação

CSS

O CSS surge para separar o conteúdo da sua aparência.

universal ✔ O seletor universal (*) aplica o estilo a todos os elementos da página.
id ✔ O atributo id permite identificar univocamente um elemento do HTML (muito útil no JavaScript).
tipo ✔ O seletor de tipo usa o nome do marcador a estilizar.

div p ❌ todos os p que estão “dentro” de um div
div > p ❌ todos os p que estão “logo dentro” de div
div ~ p ❌ todos os p que estão colocados depois de um div
div + p ❌ todos os p que estão colocados “imediatamente” depois de um div
div,p ❌ os div e p ficacarão com fonte vermelha
